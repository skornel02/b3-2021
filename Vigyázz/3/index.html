A következő képletekben a code karaktersorozat a dobozokban megjelenő bináris számra utal.

Akkor kattints a kék dobozra, ha ez az állítás igaz: code % 2 == 0

Akkor kattints a fekete dobozra, ha ez az állítás igaz: code >= 16

Akkor kattints a zöld dobozra, ha ez az állítás igaz: code - 1 == code >> 1 << 1
